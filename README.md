# Coffee-Scan
### A GUI APP that performs 3 scans in one, shows its outputs and stores them in the ``~/autoscan/`` directory.
### This app execute:

#### **1.** ``sudo nmap -sS --min-rate 5000 <IP>``

#### **2.** ``sudo nmap -sCV -p<PORTS> <IP>``

#### **3.** ``whatweb <IP>``
